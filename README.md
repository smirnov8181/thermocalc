# ThermoCalc

Теплотехнический калькулятор ограждающих конструкций. Расчёт утепления, точки росы и теплопотерь по СНиП 23-02-2003.

![ThermoCalc Screenshot](screenshot.png)

## Возможности

- **Конструктор стены** — добавление и перестановка слоёв из каталога материалов (утеплители, кладка, отделка)
- **Визуализация сечения** — пропорциональные цветные слои с подписями
- **Расчёт тепловой защиты** — сопротивление теплопередаче (R) vs нормативное значение
- **График температур** — распределение температуры в толще стены с линией точки росы
- **Анализ влаги** — оценка риска конденсата на внутренней поверхности
- **Теплопотери** — расчёт потерь в Вт/м² и кВт·ч/м² за сезон
- **Тёмная тема** — переключение light/dark mode
- **Адаптивный дизайн** — мобильные экраны от 320px, планшеты, десктоп

## Стек

Нулевые зависимости. Один HTML-файл:
- Vanilla JS
- CSS с oklch() цветами и CSS-переменными
- Canvas API для графиков

## Быстрый старт

```bash
# Просто открыть в браузере
open index.html

# Или через локальный сервер
python3 -m http.server 8080
# → http://localhost:8080
```

## Нормативная база

- СНиП 23-02-2003 «Тепловая защита зданий»
- СП 23-101-2004 «Проектирование тепловой защиты зданий»
- ГОСТ Р 54851-2011

## Дизайн-система

Монохромная палитра (oklch), шрифты Inter + JetBrains Mono, радиусы по шкале 6–24px, компоненты в стиле shadcn/ui.
